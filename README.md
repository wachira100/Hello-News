# Hello-News
This is a global news siting new events every season.
A simple news blog website showcasing various news categories including politics, technology, sports, entertainment, and more. Built with HTML and CSS.

Features
Responsive design

Multiple news categories (20+ articles)

Navigation menu with Home, Contact, and About links

Embedded images for each news article

Semantic HTML structure

File Structure
index.html - Main HTML file containing the blog structure

External image URLs used for content

How to Use
Open index.html in any modern web browser

Navigate through different news categories using the navigation menu

Click on the main logo to return to the homepage
